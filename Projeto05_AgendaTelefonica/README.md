# 📒 Agenda Telefônica em Python

Projeto desenvolvido para praticar estruturas de dados, manipulação de listas, validação de entradas e lógica de programação utilizando Python.

## Funcionalidades

- ➕ Adicionar contatos
- 📋 Listar contatos
- 🔍 Buscar contato pelo nome
- ❌ Remover contato
- ✅ Validação de DDD
- ✅ Validação de telefone
- ✅ Validação simples de e-mail
- ✅ Padronização de nomes

## Tecnologias utilizadas

- Python 3
- Regex (`re`)
- Listas
- Loops (`while` e `for`)
- Estruturas condicionais (`if`, `elif` e `else`)

## Conceitos praticados

- Manipulação de listas
- Busca em listas
- Remoção de elementos
- Validação de entradas
- Manipulação de strings
- Expressões regulares (Regex)

## Aprendizados

Durante este projeto pratiquei a criação de menus interativos, validação de dados, busca e remoção de registros em listas, além de melhorar a organização do código e a experiência do usuário.
