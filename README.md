# 🐍 Python Estudos

Bem-vindo ao meu repositório de estudos em Python!

Este repositório reúne os projetos que desenvolvo durante minha jornada de aprendizado, colocando em prática os conceitos estudados por meio de desafios e pequenos sistemas.

Meu objetivo é evoluir continuamente na programação, construindo uma base sólida em Python para posteriormente aplicar esse conhecimento em automação de testes (QA), desenvolvimento e Ciência de Dados.

---

## 📚 Projetos

### 📌 Projeto 01 - Controle de Gastos
Sistema simples para registrar despesas, calcular o total gasto e o saldo restante com base no salário informado pelo usuário.

**Conceitos utilizados:**
- Variáveis
- Listas
- Laços de repetição (`for`)
- Entrada e saída de dados
- Formatação de valores monetários

---

### 📌 Projeto 02 - Consulta de CEP
Aplicação que consulta endereços utilizando a API pública ViaCEP.

**Conceitos utilizados:**
- Consumo de API REST
- Biblioteca Requests
- Expressões Regulares (Regex)
- JSON
- Dicionários
- Validação de CEP

---

### 📌 Projeto 03 - Catálogo de Filmes
Sistema de catálogo em terminal que permite cadastrar e listar filmes.

**Conceitos utilizados:**
- Menus interativos
- Loops (`while`)
- Condicionais (`if`, `elif`, `else`)
- Listas dentro de listas
- Organização de dados

---
