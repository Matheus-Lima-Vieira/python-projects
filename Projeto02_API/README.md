# 📍 Projeto 02 - Consulta de CEP

## Objetivo

Desenvolver uma aplicação capaz de consultar endereços utilizando a API pública ViaCEP.

---

## Funcionalidades

- Receber CEP digitado pelo usuário
- Remover caracteres inválidos
- Consultar a API ViaCEP
- Exibir endereço encontrado
- Validar CEP inexistente

---

## Conceitos praticados

- Biblioteca Requests
- Consumo de API REST
- JSON
- Dicionários
- Regex (`re.sub`)
- Tratamento simples de erros

---

## Aprendizados

Neste projeto aprendi como consumir APIs utilizando Python e interpretar respostas em formato JSON.

Também pratiquei manipulação de strings utilizando Expressões Regulares para tratar a entrada do usuário.
