# 📦 Projeto 06 - Sistema de Controle de Estoque

## 📖 Sobre o projeto

Este projeto foi desenvolvido como parte da minha jornada de estudos em Python, com o objetivo de praticar manipulação de listas, lógica de programação e operações de um CRUD (Create, Read, Update e Delete).

O sistema permite cadastrar, consultar, alterar e remover produtos de um estoque, além de calcular automaticamente o valor individual de cada item e o valor total armazenado.

---

## 🚀 Funcionalidades

* ➕ Cadastro de produtos
* 📋 Listagem completa dos produtos
* 🔍 Busca por nome
* ✏️ Alteração de nome, quantidade e preço
* 🗑️ Remoção de produtos
* 💰 Cálculo do valor total por produto
* 📊 Cálculo do valor total do estoque

---

## 🛠️ Tecnologias utilizadas

* Python 3
* Expressões Regulares (Regex)
* Listas de listas
* Estruturas de repetição (`while` e `for`)
* Estruturas condicionais (`if`, `elif` e `else`)

---

## 💻 Exemplo de utilização

```text
==== ESTOQUE ====

1 - Adicionar produto
2 - Listar produtos
3 - Buscar produto
4 - Alterar produto
5 - Remover produto
6 - Sair
```

Ao listar os produtos, o sistema apresenta:

* Nome do produto
* Quantidade em estoque
* Preço unitário
* Valor total daquele item
* Valor total do estoque
